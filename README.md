# E-Commerce Sales Analysis Using SQL

## Project Overview
This project analyzes e-commerce sales data using MySQL and SQL queries.

## Technologies Used
- SQL
- MySQL

## Database Tables
- Customers
- Products
- Orders
- Order_Details

## Analysis Performed
- Total Revenue Analysis
- Top Selling Products Analysis
- Top Customer Analysis
- Revenue by Product Analysis

## SQL Concepts Used
- Primary Key
- Foreign Key
- Joins
- Group By
- Order By
- Aggregate Functions
